Programming Basics & Dynamic Memory

This section introduced me to fundamental programming concepts such as primitive data types, variables, control structures, and functions. I also learned about pointers and dynamic memory allocation, which are essential in C programming. Understanding how to manage memory efficiently can lead to significant improvements in application performance, especially in environments with limited resources.

#include <stdio.h>
#include <stdlib.h>

int main() {
    // Allocating memory dynamically for an array of integers
    int *ptr = malloc(sizeof(int) * 10);
    if (ptr == NULL) return 1; // Check for memory allocation failure
    
    ptr[0] = 5; // Assigning value to the first element
    printf("First element: %d\n", ptr[0]);
    
    free(ptr); // Freeing allocated memory
    return 0;
}

Big O Notation and Complexity

Learning algorithmic complexity taught me the importance of efficiency. Big O notation helps in estimating the performance of an algorithm relative to the size of the input. It provides a high-level understanding of how algorithms scale and allows for comparing different algorithms based on their time and space requirements. This is crucial for optimizing code and ensuring that it runs efficiently, especially in large-scale applications.

List Structures (Linked Lists)

In this section, I learned about linked lists, a fundamental data structure that consists of nodes connected through pointers. Unlike arrays, linked lists allow for dynamic memory allocation and efficient insertion and deletion operations. Understanding linked lists has helped me appreciate the trade-offs between different data structures, as they offer flexibility at the cost of random access efficiency.

#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node* next;
} Node;

void insert(Node** head, int newData) {
    Node* newNode = (Node*) malloc(sizeof(Node));
    newNode->data = newData;
    newNode->next = (*head);
    (*head) = newNode;
}

// Function to print the linked list
void printList(Node* node) {
    while (node != NULL) {
        printf("%d -> ", node->data);
        node = node->next;
    }
    printf("NULL\n");
}

Stacks and Queues

Stacks and queues are essential data structures that represent collections of elements with specific access rules. Stacks follow the Last In, First Out (LIFO) principle, while queues operate on a First In, First Out (FIFO) basis. Learning about these structures has enabled me to understand various algorithms and their applications, such as depth-first and breadth-first search.

I implemented a function to check if a string is a palindrome using a deque (double-ended queue). This approach allows for efficient insertion and removal of characters from both ends, making it easier to compare characters from the front and back of the string

Trees and Hashing

Trees are hierarchical data structures that allow for efficient searching, inserting, and deleting operations. Hashing, on the other hand, provides a method to access data in constant time through the use of hash functions. Understanding these structures is vital for handling large datasets and ensuring optimal performance in applications.

Graphs

Graphs are versatile data structures used to represent relationships between objects. They can be directed or undirected, weighted or unweighted. Learning about graphs has opened my eyes to various real-world applications, such as network routing and social network analysis.
